maidtyan
========
メイドちゃんは、スケジュール管理をしてくれます。    

ruby maid-tyan.rb で スケジュールメールを送信します。  


■設定方法
Googleアカウントが必要になります。    
config.rb にアカウント情報や送信先アドレスなどを記入して、    
メイドちゃんに渡すメールを gmailでlabel付けしてください。    


また、いくつかのgemが必要になります。    
gem install ruby-gmail    
gem install gcalapi    
gem install weather_hacker
